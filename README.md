# genqr-srv
qrcode generate service use deno

## Install Deno
https://deno.land/#installation

## Start Serve
deno run --allow-net --allow-read --unstable index.ts

## Dev Mode, Auto reload serve
deno run --watch --allow-net --allow-read --unstable index.ts
